Tips and Tricks

Change repository or image name

Scenario: You have a REPOSITORY named old_image and tagged old_tag. 
REPOSITORY          TAG                 IMAGE ID            CREATED             VIRTUAL SIZE
old_image           old_tag             a1b2c3d4e5f6        1 minutes ago       63.5 MB
You want to rename it to new_image and tag new_tag. 
docker tag a1b new_image:new_tag
where a1b is the first 3 characters of the IMAGE ID. You can ultimately delete the old image using 
docker rmi old_name

push a local docker image to your dockerhub account

docker images
gives 
REPOSITORY          TAG                 IMAGE ID            CREATED             VIRTUAL SIZE
local_image         local_tag           a1b2c3d4e5f6        1 minutes ago       63.5 MB

docker tag a1b2c3d4e5f6 dockerhub_username/dockerhub_image:dockerhub_tag
docker push dockerhub_username/dockerhub_image:dockerhub_tag
where dockerhub_tag may or may not be the same as the local_tag. The same is true about local_image and dockerhub_image.

saving the local image

docker save local_image > local_image.tar
and later load it using 
docker load --input local_image.tar
